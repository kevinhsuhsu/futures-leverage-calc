# 台股期貨槓桿計算機

純前端（無後端、無 build、無 toolchain）的台股期貨槓桿／保證金／風險試算工具。資料存瀏覽器 localStorage，單機保存。

兩個分頁：

- **多檔持倉**：管理你的庫存，看每檔與整體的槓桿、保證金、權益數與風險。
- **單品試算**：下單前規劃單一商品該買幾口、不同槓桿的風險（仿 alpha.futures-ai.com/leverage-simulator）。

## 跑起來

用到 `fetch` 載入 `data/multipliers.json`（ES module），**必須用 http server 開，不能直接 `file://`**：

```bash
cd ~/Code/futures-leverage-calc
python3 -m http.server 8000
# 開 http://localhost:8000
```

## 多檔持倉

- **更新保證金**：一鍵從期交所抓全部商品保證金存 local（手動觸發、低頻，避免黑名單）。1 小時內再按沿用快取；proxy 掛掉時用「手動貼上」備援。
- **商品搜尋**：依代碼／名稱／標的，分類（指數／個股／ETF），點「加入」建庫存。
- **庫存 CRUD**：口數／成本即時改、多空切換（多=紅、空=綠）、刪除。
- **每列欄位**：契約價值、原始／維持保證金、槓桿、每跳損益、漲停損益、跌停損益；開即時後加 合約代碼／現價／未實現損益。底部「帳戶合計」列加總契約價值／保證金／槓桿／未實現損益。
- **權益數（模擬期貨商帳戶）**：期初權益（本金）、浮動損益、權益總值、操作槓桿、風險等級、原始／維持保證金、保證金使用率、可動用（超額）保證金、風險指標、距追繳金額、距斷頭金額。低於維持保證金→追繳警告；風險指標 < 25%→斷頭警告。
- 卡片可收合；庫存 JSON 匯出／匯入（換機備份）。

## 單品試算

輸入帳戶權益、商品（可搜尋）、收盤／進場價、口數、承受回檔%，輸出：

- **原始商品合約規格**：契約價值、原始保證金、每點價值、商品槓桿、漲停價、跌停價。
- **帳戶操作合約規格**：總契約價值、使用保證金（含使用率）、每點價值、操作槓桿。
- **自訂風險承受能力**：依承受回檔% → 回檔上限點／%、預估損失金額、維持保證金。
- **最大風險承受能力**：低於維持保證金 點／%、超額損失 點／%。
- **目前槓桿** + **槓桿對照表（10/20/30x）**：各槓桿對應口數／使用率／回檔上限。

公式以 reference 數字反推驗證（商品槓桿、操作槓桿、回檔／低於維持／超額點數、對照表口數皆吻合）。

## 兩種保證金制度（核心）

| 類別 | 制度 | 槓桿 |
|---|---|---|
| 個股期貨 | 百分比（原始／維持為契約價值比例） | 1 / 原始比例（與價格無關） |
| 指數 / ETF 期貨 | 固定金額（每口 NTD） | 契約價值 / 原始保證金（需成本×乘數） |

- 個股 1 口＝2000 股（小型 100 股）。
- 指數／ETF 算槓桿需**契約乘數**，期交所頁不提供 → 內建 `data/multipliers.json`（大台 200／小台 50／微台 10／電子 4000／金融 1000／非金電 100…）；未列者乘數留空，加入庫存時可手動覆寫。
- **每跳損益** = 最小跳動 × 每點價值 × 口數；個股／ETF 依台股 tick 表隨價分級，指數依商品（台指 1 點、電子 0.05、金融 0.2）。
- **風險指標** = 權益總值 ÷ 原始保證金 ×100%，< 25% 多數期貨商盤中代為沖銷（斷頭）；權益總值低於維持保證金則盤後追繳。
- **操作槓桿** = 部位市值 ÷ 權益；**風險等級**：<2x 保守、2–5x 穩健、5–10x 積極、10–20x 高風險、≥20x 極高風險。

## 即時報價（Fugle，選配）

權益數區塊勾「Fugle 即時報價」+ 貼 [Fugle 金鑰](https://developer.fugle.tw/docs/data-futopt/intro/)，每檔填**近月完整合約代碼**（加入時下拉預設近月）。

- 端點：`https://api.fugle.tw/marketdata/v1.0/futopt/intraday/quote/{symbol}`，header `X-API-KEY`。實測 Fugle **允許瀏覽器直連**（無 CORS 問題）。
- 合約代碼：根代碼 + 月碼（**A–L = 1–12 月**）+ 西元末碼，如 7 月 2026 大台 = `TXFG6`。近月過了結算日（每月第 3 個週三）自動跳次月。
- 取得現價後計算未實現損益、權益總值、風險指標、距追繳／斷頭。

## 已知限制

- 保證金為**單式相加，未計入 SPAN 組合保證金抵減** → 實際所需可能更低。
- 不含股票／ETF 選擇權。
- 公開 CORS proxy（allorigins／corsproxy）是更新保證金的唯一外部相依，掛掉用「手動貼上」。
- 保證金每季調整，逾 90 天未更新會提示。
